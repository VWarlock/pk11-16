
                      17. Калькулятор
							  17.09.90

     Калькулятор представляет собой процесс,  предназначенный  для 
выполнения арифметических вычислений.  Калькулятор  размещается  в 
файле Calc.Prc и обычно запускается в  стартовом  командном  файле
утилитой Prun.
     После запуска калькулятор находится  в  пассивном  состоянии. 
Для  того,  чтобы  активировать  калькулятор,  необходимо  набрать 
комбинацию Shift+СБРОС. При этом окно с калькулятором переместится
на  передний  план  и   можно   начинать   вычисления.   Перевести 
калькулятор в пассивное состояние можно клавишей АР2.
     Табло калькулятора состоит из двух  строк.  В  первой  строке 
пользователь набирает арифметическое выражение, значение  которого 
необходимо  вычислить.  Во  второй  строке   выводится   результат 
вычисления.
     При наборе  выражения  необходимо  пользоваться  синтаксисом, 
аналогичным  языку  программирования  Бейсик.  В  выражении  можно 
использовать функции, обозначенные на клавишах  калькулятора.  Для 
набора функции достаточно нажать на клавиатуре букву, выделенную в 
названии функции красным цветом.  Для  использования  в  выражении 
результата   предыдущего   вычисления   необходимо   поставить   в 
соответствующее место выражения букву R.
     Для вычисления значения набранного выражения нажмите  клавишу 
═ или ВК. Во второй строке будет выведен резульатат  вычисления  в 
10─тичной, 8─ричной  и  16─ричной  системах  счисления.  Курсор  в 
строке выражения переместится в  первую  позицию.  Можно  набирать 
новое выражение или отредактировать предыдущее, используя  клавиши 
движения курсора ← и →.
     По умолчанию при вводе чисел используется  10─тичная  система 
счисления. При  вводе  числа  в  восьмеричной  системе  необходимо 
использовать префикс #, а при вводе 16─ричного числа ─ префикс  $, 
например:
     #177514   ─ 8─ричное число,
     $BD16     ─ 16─ричное число.
