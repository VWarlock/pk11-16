на  языке  MML  используются  следующие односимвольные команды:

   A-G - нота (С - до, D - ре, E - ми, F - фа, G - соль, A - ля,
B - си) в текущей (предварительно выбранной) октаве;
   R - задает музыкальную паузу;
   + - увеличивает высоту звучания ноты (за которой следует +)
на полтона; соответствует диезу;
   - - уменьшает высоту звучания ноты (за которой следует -) на
полтона; соответствует бемолю;
   . - увеличивает длительность ноты на половину; после ноты
может указываться несколько точек;
   On - устанавливает в качестве текущей октаву n; всего
доступны октавы с номерами 1-7 (по умолчанию O4);
   Ln  -  задает  длительность  для  следующих  нот;  фактическая
длительность определяется как 1/n: L1 - целая нота,  L64  -  одна
шестьдесят  четвертая  ноты  длительность  ноты  может  следовать
непосредственно за  самой  нотой;  (A16  эквивалентно  L16A);  по
умолчанию - длительность L4;
   Tn  -  задает  темп  проигрывания  мелодии;  n  -   количество
четвертей  в  минуту;  может  принимать   значения   32-255;   по
умолчанию значение n принимается равным T120;
   Vn -  задает  громкость  звучания;  может  принимать  значения
1-16; по умолчанию используется значение V8;
   Mn - задает период огибающей в тиках (т.е.  1/50  секунды);  n
может принимать значения 1-255;
   Sn - задает номер формы  огибающей;  допустимые  значения  для
номера огибающей 1-7; по умолчанию не  используется  ни  одна  из
огибающих;
